PageWrapper

PageWrapper class

A widget that wraps the main content of a page.

This widget is a stateless widget that takes a title, body, bottom navigation bar, menu actions, header controls, padding, and a flag for simple design and loading screen override as input. It displays a Scaffold with an AppBar, body, and bottom navigation bar. The AppBar contains the title and an optional action menu. The body contains the main content of the page, wrapped in a Container with padding.
