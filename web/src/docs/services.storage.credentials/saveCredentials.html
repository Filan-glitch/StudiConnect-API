Saves the user's credentials in secure storage.

The userID and sessionID parameters are required. This function does not return a value.

userID is the unique identifier for the user. sessionID is the unique identifier for the current session.

Implementation

Future<void> saveCredentials(String userID, String sessionID) async {
  final FlutterSecureStorage storage = secureStorage;
  log('Saving credentials');
  await storage.write(key: 'userID', value: userID);
  log('Saved userID');
  await storage.write(key: 'sessionID', value: sessionID);
  log('Saved sessionID');
}
